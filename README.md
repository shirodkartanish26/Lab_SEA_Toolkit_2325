COVID-19 Country-Wise Data Analysis

This project analyzes the country_wise_latest COVID-19 dataset using R as part of the Data Science and Toolkit course.
The main objective is to explore global COVID-19 patterns through visualisation and derive meaningful insights from the data.

The analysis includes:

Identifying the top 10 countries with the highest confirmed cases

Identifying the top 10 countries with the highest deaths

Understanding global share of cases and deaths using pie charts

Exploring distribution of confirmed cases using a histogram

Generating findings that highlight uneven global impact

The project uses tidyverse, ggplot2, and base R plotting tools.
